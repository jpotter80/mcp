## Why Archived?

These documents represent the project's evolution from a single-purpose Mojo documentation server to a multi-server architecture framework. They've been superseded by:

- **Current documentation** in `/docs/` directory
- **Consolidated architecture** in `docs/ARCHITECTURE.md`
- **Updated guides** reflecting the final multi-server design

## Historical Value

These documents are preserved for:

- Historical context for design decisions
- Learning from the restructuring process
